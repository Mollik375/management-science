Project 5: Linear Programming

📈 Project 5: Linear Programming

📖 Project Overview

Linear Programming is used to maximize or minimize an objective function subject to constraints.

🎮 Demonstration

Objective: Maximize Profit

Profit = 50X + 30Y

Constraint 1: X + Y ≤ 100

Constraint 2: X ≤ 60

Result: Optimal production occurs at the feasible corner point.
